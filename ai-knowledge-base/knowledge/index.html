2. PageIndex
⭐ 30.6K 🐍 Python rag framework
无向量库、基于推理的 RAG 系统，通过构建文档的层级树状索引，用 LLM 推理替代向量相似度搜索进行检索。在 FinanceBench 上达到 98.7% 准确率。
亮点：颠覆传统 Vector RAG 范式，用推理替代相似度搜索，理论优雅且效果惊艳（98.7% 准确率）。
评分理由：RAG 领域的重要创新，摆脱了向量数据库依赖，用树搜索+推理实现更精准的检索，可能是下一代 RAG 架构方向。
3. local-deep-research
⭐ 7.2K 🐍 Python rag framework llm
全本地 AI 深度研究系统，支持本地 LLM（Ollama/llama.cpp）和云模型，集成 10+ 搜索引擎（arXiv/PubMed/Wikipedia 等），SimpleQA 达 ~95%，数据 AES-256 加密，零遥测。
亮点：首个在单张 RTX 3090 上实现 ~95% SimpleQA 的开源项目，隐私优先（全程本地运行 + 数据加密）。
评分理由：本地深度研究领域的标杆项目，性能优秀且完全离线可运行，对隐私敏感用户和研究机构价值巨大。
